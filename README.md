Contact List - JS Mini Project

A simple Contact List web application built with HTML, CSS, and JavaScript. This project allows you to add, search, and delete contacts, and saves them in your browser using localStorage.

Features

Add new contacts with name, phone number, and email

Search contacts by name

Delete individual contacts

Clear all contacts

Data is saved in your browser so it persists on refresh

Responsive design for desktop and mobile

Technologies Used

HTML5

CSS3

JavaScript (ES6+)

How to Use

Open index.html in your browser.

Fill out the form and click Add contact.
